go基础（三）数据类型

Tue, Mar 7, 2023

详情看官网https://go.dev/doc/effective_go#data

数据类型总览

序号 类型 描述
1 Bool true or false
2 int int float32 float64
3 string Go 的字符串是由单个字节连接起来的。go语言字符串使用Utf-8编码标识Unicode文本
4 派生类 指针（Pointer）数组结构体（struct）通道（Channel）函数切片接口（interface）Map

数字类型

序号 类型 描述
1 uint8 无符号8位整型（0-255）
2 uint16 无符号8位整型（0-65535）
3 uint32 无符号8位整型（0-4294967295）
4 uint64 无符号8位整型（0-18446744073709551615）
5 int8 无符号8位整型（-128-127）
6 int16 无符号8位整型（-32768-32767）
7 uint32 无符号8位整型（-2147483648-2147483647）
8 uint64 无符号8位整型（-9223372036854775808-9223372036854775807）

浮点型

序号 类型 描述
1 float32 IEEE-754 32位浮点型数
2 float64 IEEE-754 64位浮点型数
3 complex64 32位实数和虚数
4 complex128 64位实数和虚数｜

其他数字类型

序号 类型 描述
1 byte 类似uint8
2 rune 类似int32
3 uint 32或64位
4 int 与nint一样
5 uinptr 无符号整型，用于存放一个指针

参考https://www.runoob.com/go/go-data-types.html

bool类型

  • 不能使用0 和非0表示bool类型
  • 只有true和false
  • bool值与数字的转换 
    func itob(i int) bool { return i != 0 }

字符串类型

  • 单行：双引号扩起来

  • 多行：反引号扩起来

  • 字符串连接方式

    • +号 
      s1 := "hello"
s2 := " world"
fmt.Println(s1 + s2)

    • s1 := "hello"
s2 := " world"
fmt.Printf("%s %s", s1, s2)

    • s1 := "hello"
s2 := " world"
s3 := strings.Join([]string{s1, s2}, " ")
fmt.Println(s3)

    • s1 := "hello"
s2 := " world"
var buffer bytes.Buffer
buffer.WriteString(s1)
buffer.WriteString(" ")
buffer.WriteString(s2)
fmt.Printf("buffer.String(): %v\n", buffer.String())
