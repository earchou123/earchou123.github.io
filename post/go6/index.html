数组

数组定义

var 数组[数组大小]数据类型

数组长度固定,可以由0个或多个元素组成

var arr1 = [3]int{1,2,3}

var arr2 [...]int //数组不确定长度，由...代替，长度由初始化的数据决定
arr2 = [1,2,3,4,5]

  //指定所引值初始化
a := [...]int{1: 1, 3: 5} //[0,1,0,5]

数组的比较：当两个数组的值和类型都相等，才相等

a := [2]int{1, 2}
b := [...]int{1, 2}
c := [2]int{1, 3}
d := [2]int{2, 1}
fmt.Println(a == b, a == c, a == d) //true false false

数组的内存布局

  • 数组的地址通过数组名来获取 &arr
  • 首元素地址就是数组的首地址
  • 数组个个元素地址间隔是依据数组的类型来决定的，如int -> 8个字节

Tips

  • 数组是值传递，支持“==”，“！=”等操作符
  • [n]*T表示指针数组，*[n]T表示数组指针
  • 多维数组只有第一层可以使用…来让编译器推导数组长度
